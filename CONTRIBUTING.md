# Contributing

This repository is currently **private**.

## Ground rules

1. No live secrets, tokens, or credentials.
2. No personal machine inventory that is not needed for the guide.
3. Prefer placeholders from `docs/PLACEHOLDERS.md`.
4. Keep the testing disclaimer accurate.
5. If you change install commands, note the date and what you tested.

## Suggested PR / update checklist

- [ ] Guide still works stage-by-stage for a less-technical reader
- [ ] Placeholders remain consistent
- [ ] Disclaimer still matches actual test coverage
- [ ] No absolute personal paths unique to one human slipped in
- [ ] README repo title/status still correct if renaming for public launch
